## Instructions:
1. Place the 'Herts2u' folder in `C:\xampp\htdocs\`  
2. Import the database:  
   - Open phpMyAdmin  
   - Create a new database (e.g., `herts2u_db`)  
   - Click **Import** and upload `herts2u_db.sql`  
3. Run the project at: [http://localhost/Herts2u](http://localhost/Herts2u)  

---

## Features
- User registration and login with secure password hashing  
- Session management for user privacy  
- Frontend journaling interface (currently static; no backend save yet)  
- Responsive design for multiple devices  

---

## How to Run
1. Install XAMPP or a similar local server environment.  
2. Place the project folder inside the `htdocs` directory.  
3. Import the database using the provided `.sql` file via phpMyAdmin.  
4. Start Apache and MySQL in XAMPP.  
5. Access the website via `http://localhost/Herts2u`
